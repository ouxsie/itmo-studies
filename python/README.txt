В папке Basics-of-Python собраны относительно типовые задачи по визуализации данных, анализу с помощью pandas, а также интерфейсы с помощью tkinter. 

В папке DDDM (Data-Driven Decision Making) лежит наш проект по аналитике антикоррупционных ведомостей российских чиновников. Есть презентация и код для второй гипотезы, которую я анализировал.

В папке HCI (Human-Computer Interaction) лежит лишь один проект - Holiday for Every Day. Он был создан с Машей Ливицкой и Дианой Андреевой. Микро-приложение для праздников на сегодня ±2 дня, использован объектно-ориентированный подход и парсинг сайтов с requests и BeautifulSoup.

___
Basics-of-Python
This folder contains relatively standard tasks covering data visualization, pandas-based analysis, and tkinter interfaces.

DDDM (Data-Driven Decision Making)
Here you can find our project on anti-corruption delarations of the Russian bureaucracy. There's a presentation and a .py with the analysis of H2 I preformed.

HCI (Human-Computer Interaction)
This directory includes a single project: "Holiday for Every Day" — a micro-app that displays holidays for today ±2 days. It was created with Masha Livitskaya and Diana Andreeva. Built using an object-oriented approach, it employs web scraping with requests and BeautifulSoup.
