Тут собраны проекты по SQL. 

gunstore - проект базы данных для оружейного магазина. Внутри в ворд-файле лежит подробное описание всех сущностей, отношений и предметной области, а также ER-диаграмма.

deealership - проект базы данных для выдуманного автосалона, сделанный совместно с Лизой Третьяк. В папке reports лежат подробные отчёты с DF-, IDEF1X- и ER-диаграммами (ЛР 1-2). Также была предпринята попытка переноса БД на нереляционную MongoDB (ЛР5).

The reports folder contains detailed documentation with DF, IDEF1X, and ER diagrams (Labs 1–2). Additionally, an attempt was made to migrate the database to a non-relational MongoDB (Lab 5).
